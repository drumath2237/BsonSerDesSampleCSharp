# BsonSerDesSampleCSharp

## About

C#でBSONデータをシリアライズ/デシリアライズするサンプル（コンソールアプリ）です。

## Environment

- Windows 11 Pro
- .NET 7 (Console App)
- Newtonsoft.Bson 1.0.3-beta

## Usage

```bash
# build
dotnet build

# run app
dotnet run --project BsonSerDesSampleCSharp
```
